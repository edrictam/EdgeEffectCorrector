This repository contains the code that implements the edge effect corrector considered in the manuscript SPARK: in-silico simulations for benchmarking nascent RNA sequencing experiments. 

The simulator portion of the code can be found in the repository https://github.com/thepailab/spark. 


Reference: 

SPARK: in-silico simulations for benchmarking nascent RNA sequencing experiments

Ezequiel Calvo-Roitberg, Jesse W. Lehman, Edric Tam, Shaimae Elhajjajy, Barbara E. Engelhardt, Athma A. Pai
